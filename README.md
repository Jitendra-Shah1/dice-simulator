#  Dice Simulator

A simple Python project that simulates rolling a dice and stores the results in an SQLite database.  
It allows you to roll dice, view statistics of past rolls, and identify the most frequent number rolled.

---

## Project Structure
- dice.py → Main program file containing all logic.
- dice.db → SQLite database file created automatically to store dice rolls.

---

##  Features
- Roll a dice (values between 1–6).
- Store each roll in a database.
- Show statistics:
  - Total rolls
  - Count and percentage of each number rolled
  - Most frequent number
